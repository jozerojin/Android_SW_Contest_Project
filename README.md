# 동적 QR코드를 이용한 인증 시스템
(Dynamic QR Code-based Authentication System)

교내 SW 프로젝트 경진대회 출품작으로,  
주기적으로 변경되는 QR 코드와 추가 인증 번호 입력을 결합한 **이중 보안 인증 시스템**입니다.



## 프로젝트 개요

본 프로젝트는 정적인 QR 코드 인증 방식의 보안 취약점을 개선하기 위해  
**동적 QR 코드 + 추가 인증 번호 입력**을 결합한 인증 방식을 구현하였습니다.

QR 코드 이미지와 내부 데이터는 주기적으로 변경되며,  
QR 코드 스캔 이후에도 별도의 인증 번호를 입력해야 인증이 완료되는 구조로  
QR 코드 재사용 및 캡처 기반 공격을 방지하는 것을 목표로 합니다.



## 시스템 구성

본 시스템은 **Android 앱 2종과 Node.js 서버**로 구성되어 있습니다.



## 주요 기능

- 동적 QR 코드 생성 및 주기적 갱신
- QR 코드 스캔 기능 제공
- 추가 인증 번호 입력을 통한 이중 인증
- 서버 기반 QR 데이터 및 인증 정보 검증
- REST API 기반 클라이언트–서버 통신



## 구성 요소별 역할

### client_rental
- 사용자 측 안드로이드 애플리케이션
- QR 코드 스캔 및 인증 번호 입력
- 인증 요청 전송 및 결과 수신

### vd_create
- 인증에 사용되는 동적 QR 코드 생성 전용 앱
- 서버로부터 QR 데이터 수신 및 QR 이미지 생성

### server
- Node.js 기반 인증 서버
- QR 코드 생성 및 검증 로직 처리
- 인증 토큰 및 내부 데이터 관리
- Android 클라이언트와 REST API 통신



## 인증 흐름

1. vd_create 앱이 서버에 요청하여 동적 QR 코드 데이터 생성
2. QR 코드 이미지 및 내부 데이터가 주기적으로 갱신됨
3. client_rental 앱에서 QR 코드 스캔
4. 사용자로부터 추가 인증 번호 입력
5. 서버에서 QR 데이터와 인증 번호를 함께 검증
6. 인증 결과를 client_rental 앱으로 반환
